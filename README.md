# what's access
access in a reverse proxy built on top of node js

## how to use it 
create a ```config.json``` file. then start your configutation <br>

``` 
{
        "domain name": {
            "host": "local ip adress and port"
        },
}
```

<hr>

exemple:
```json
    {
        "exemple.com": {
            "host": "http://127.0.0.1:3000"
        },
    }

```
